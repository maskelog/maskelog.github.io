Hugo로 시작하는 Github.io

개발공부 기록을 할 블로그 서비스를 찾아보다가 주소가 github.io로 끝나는 블로그들이 있어서 찾아보니 Github와 오픈소스 정적 사이트 생성 서비스를 사용하여 빠른 웹페이지 생성이 가능하다고 합니다.

오픈소스 정적 사이트 생성 서비스는 Jekyll, Hexo, Gatsby, Hugo, VuePress 등이 있습니다.
go언어를 사용한 Hugo가 궁금해서 테스트를 해봤습니다

Hugo 설치 방법 (윈도우11 기준)

https://gohugo.io/installation/windows/

winget install Hugo.Hugo.Extended
hugo version

powershell을 열고 winget 명령어만 입력하면 간단하게 설치가 가능하고 환경변수도 자동으로 설정되었다 이후 정상적인 설치가 되었는지 확인 하는 명령어를 입력하였다

github에 접속해서 repo 이름이 blog와 사용자ID.github.io 레포지토리를 생성한다.
powershell을 열고 원하는 디렉토리에 이동 후 명령어를 입력해 hugo 사이트를 생성한다.

hugo new site blog 
// blog 이름의 hugo site 폴더 생성

테마 설정하는 법

https://themes.gohugo.io/ 적용방법은 테마별로 다르다 상세항목을 참조 바란다. 이 블로그는 m10c테마를 사용하였다


m10c 테마 기준으로 설명하겠다

powershell을 열고 hugo site를 생성한 폴더로 이동한다

git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

git clone 이후 테마를 적용하기 위해서는 hugo.toml파일을 수정해야한다.
hugo.toml는 Hugo 정적 사이트 생성기에서 사용되는 프로젝트 설정 파일입니다.

테마를 적용하기 위해서는 hugo.toml 파일에 theme = "m10c" 을 추가합니다.

[[menu.main]]
  identifier = "tags"
  name = "Tags"
  url = "/tags/"

[[params.social]]
  icon = "github"
  name = "My Github"
  url = "https://github.com/사용자 ID"

[[params.social]]  
  icon = "twitter"
  name = "my twitter"
  url = "https://twitter.com/사용자 ID"

초기 세팅을 위해서 이러한 코드도 삽입합니다

로컬에 파일 생성 완료!

github 연결하기

github 연결 과정에서 오류가 발생하여 레포지터리 삭제만 6번정도 하였다 ㅠㅠ 오류를 해결 하고 정상적으로 github와 연동을 하였다

git remote add origin git@github.com:maskelog/blog.git
// Hugo 설정 연동
git submodule add -b master git@github.com:maskelog/maskelog.github.io.git public
// public은 블로그 페이지

이렇게 하면 github 연동이 완료되었다.

블로그 글 업로드

hugo -t 테마이름 명령을 통해 테마가 적용된 블로그 내용을 public에 생성 public 디렉토리로 이동하여

git add . // 수정된 파일들을 index에 올린다.  
git commit -m "commit message" // commit 
git push origin main commit을 사용자ID.github.io에 푸시  
blog 저장소에도 변경내용 push 하기  
cd blog  
git add .  
git commit -m "commit"  
git push origin main

블로그 업로드 확인

사용자ID.github.io로 접속을 하면 Hugo 테마가 적용된 블로그를 확인할수 있다.
